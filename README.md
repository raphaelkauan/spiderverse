## Description

An api inspired by the spiderverse with the aim of registering several spider men and their other villains.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
